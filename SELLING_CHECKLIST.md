# Selling Checklist

Checklist ini untuk memastikan template aman dan rapi sebelum dijual ke klien.

## Sebelum Menawarkan

- Pastikan frontend dan backend lokal bisa jalan.
- Ganti semua data contoh di `index.html`.
- Ganti foto, video, musik, rekening, alamat, dan Google Maps.
- Login dashboard dengan akun Anda sendiri, bukan akun demo.
- Pastikan `data-url` dan `data-key` mengarah ke backend milik Anda.

## Sebelum Serah Terima ke Klien

- Buat kredensial admin baru khusus klien.
- Ganti password admin bawaan.
- Regenerate `access_key` dari dashboard.
- Cek komentar, RSVP, dan download data berjalan normal.
- Uji tampilan di desktop dan mobile.
- Uji tombol maps, copy rekening, copy alamat, dan Google Calendar.

## Operasional Template Master

- Simpan folder ini sebagai template master.
- Saat ada order baru, duplikasi folder master.
- Isi data klien di salinan proyek, bukan di master.
- Simpan kredensial admin dan domain tiap klien secara terpisah.
